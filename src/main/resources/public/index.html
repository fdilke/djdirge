Muzurphus REST API

This API provides endpoints to:

  • query stuff: 
    curl -X GET --header \
'Accept: application/json' 'http://muzurphus.herokuapp.com/api/felixStats'
  • retrieve stuff -- for example: 
    curl -X GET --header \
'Accept: text/plain' 'http://muzurphus.herokuapp.com/api/reverse?inputString=Hello%20Felix'

See the Swagger UI for details and to try out the endpoints manually.
